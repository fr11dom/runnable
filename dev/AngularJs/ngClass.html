ngClass

  • 作用： 动态添加CSS类名。
  • 用法： 1、作为属性(attribute);2、作为类名(class)。

作为属性：<any ng-class="{expression}">…… <any>

例1：映射语法(Map) ng-class="{class: expression}"
√上checkbox看变化 红色的 蓝色的 蓝色的 删除线 红色的 红色的

源码：

"HTML:"
<input ng-model="expression" type="checkbox"/>
<span ng-class="{red: expression}">√上checkbox看变化</span>
<span ng-controller="Demo1Ctrl" ng-class="{red: color == true}">红色的</span>
<span ng-controller="Demo1Ctrl" ng-class="{blue: style}">蓝色的</span>
<span ng-controller="Demo1Ctrl" ng-class="{ blue: { {style}} }">蓝色的</span>
<span ng-controller="Demo1Ctrl" ng-class="style2">删除线</span>
<span ng-repeat="item in [red]" ng-class="{red: item == red}">红色的</span>
<span ng-controller="Demo1Ctrl" >
    <span ng-repeat="item in data" ng-class="{ {item}}">红色的</span>
</span>
"JS:"
function Demo1Ctrl($scope){
    $scope.color = true;
    $scope.style = true;
    $scope.style2 = {strike: true};
    $scope.data = [{red: true}];
}

注意：

在一个指令中调用其scope里的值,'{ {} }'是可选的。

比如: ng-class="{blue: style}"写成ng-class="{ blue: { {style}} }也是正确的"

在一个标签上调用其指令ng-repeat里的key,直接用key即可，用{key}或者{ {key}}都是有问题的

例2：映射语法(Map) ng-class="{class: expression,class2: expression2}"
√上checkbox看变化

源码：

<span ng-class="{bold: expression1, red: expression2}">√上checkbox看变化</span>
<input ng-model="expression1" type="checkbox"/>
<input ng-model="expression2" type="checkbox"/>
例3：字符串(string) ng-class="string"
输入你想加的样式，样式里提供的有bold、red等

源码：

<span ng-class="expression3">输入你想加的样式，样式里提供的有bold、red等
<input ng-model="expression3" type="checkbox"/>
例4：字符串数组(Array) ng-class="[class1, class2]"
输入你想加的样式，样式里提供的有bold、red等

源码：

<span ng-class="[expression4, expression5]">输入你要加的样式，样式提供的有bold、red</span>
<input ng-model="expression4" type="input" placeholder="可以输入bold，red等"/>
<input ng-model="expression5" type="input" placeholder="可以输入bold，red等"/>
