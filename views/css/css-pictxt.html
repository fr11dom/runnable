图文混排在各种类型的网站中都能见到，因此这道笔试题看起来确实是在考察一个前端在页面布局上的基本能力。

我起初也是这样认为，直到在@kejun博客里看到近期面试感受这篇文章。

常规的做法就不提了，看看@kejun的做法

html:

	<div class="item">
	    <div class="pic">...</div>
	    <div class="content">...</div>
	</div>

css:

.item .pic { float:left;margin-right:10px; }
.item .content { overflow:hidden;zoom:1; }

嗯，猛一看overflow:hidden是干嘛的？不正是用来触发BFC的么，这篇Block Formatting Context有了实践。
